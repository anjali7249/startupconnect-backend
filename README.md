# StartupConnect Backend

Backend API for StartupConnect platform using Node.js, Express, and Neon PostgreSQL.

## Technologies
- Node.js
- Express
- PostgreSQL (Neon)
- CORS

## API Endpoints
- GET /api/startups
- GET /api/users
- GET /api/interests
- POST /api/login

## Environment Variables
- DATABASE_URL - Neon PostgreSQL connection string
